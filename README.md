# Content Compass V2.0

Blue Soulful Director build for GitHub Pages.

## New in V2.0
- Quick Mode / Full Mode
- Location presets for recurring days
- Video / Photo Balance dropdown retained in Quick Mode
- Expanded soulful writing library
- Daily variation for hooks and lines
- Light tip banner at top of output
- Shoot checklist-only field view with script cues
- Send to Notes via Web Share API with clipboard fallback
- Archive cap raised to 90, warning at 75, and archive export
- Visible validation for 15–90 second video length

## Deploy
Upload all files to your GitHub Pages repository root, commit, then hard refresh the app on your phone/browser.
